Ringkasan Instalasi Server di Debian 12

DHCP Server

Pengertian: Server yang memberikan alamat IP secara otomatis ke klien dalam jaringan.

Perintah Instalasi: apt install isc-dhcp-server

Aplikasi: isc-dhcp-server

FTP Server

Pengertian: Server untuk mentransfer file antar komputer melalui jaringan.

Perintah Instalasi: apt install proftpd

Aplikasi: proftpd

Remote Server

Pengertian: Server yang memungkinkan kontrol komputer dari jarak jauh.

Perintah Instalasi: apt install openssh-server

Aplikasi: OpenSSH

File Server

Pengertian: Server yang digunakan untuk menyimpan dan membagikan file.

Perintah Instalasi: apt install samba

Aplikasi: Samba

Web Server

Pengertian: Server yang menyajikan halaman web ke pengguna melalui browser.

Perintah Instalasi: apt install apache2

Aplikasi: Apache2

DNS Server

Pengertian: Server yang menerjemahkan nama domain menjadi alamat IP.

Perintah Instalasi: apt install bind9

Aplikasi: BIND9

Database Server

Pengertian: Server yang menyimpan dan mengelola data dalam bentuk basis data.

Perintah Instalasi: apt install mariadb-server

Aplikasi: MariaDB atau Mysql

Mail Server

Pengertian: Server yang mengelola pengiriman dan penerimaan email.

Perintah Instalasi: apt install postfix

Aplikasi: Postfix
